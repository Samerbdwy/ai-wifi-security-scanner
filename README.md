# 📡 AI Wi-Fi Security Scanner

An intelligent wireless network security tool that scans nearby Wi-Fi networks, detects rogue access points, analyzes encryption types, and provides AI-powered security insights.

---

## 🎯 What It Does

Scan your nearby Wi-Fi networks → AI instantly tells you:

- **NETWORKS FOUND:** How many access points are nearby
- **ROGUE APS DETECTED:** Identifies honeypots, evil twins, and insecure networks
- **ENCRYPTION ANALYSIS:** WPA2/WPA3/WEP/Open classification
- **SECURITY RECOMMENDATIONS:** Actionable steps to protect your wireless environment

Works on any Windows network with Wi-Fi enabled.

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Backend** | Python + Flask |
| **AI Engine** | Google Gemini 2.5 Flash |
| **Wi-Fi Scanning** | Windows netsh commands |
| **Frontend** | HTML5, CSS3, JavaScript |
| **Security Analysis** | Pattern recognition, encryption detection |

---

## 📊 Features

| Feature | Description |
|---------|-------------|
| **Real-Time Scanning** | Detects all nearby Wi-Fi networks in seconds |
| **Rogue AP Detection** | Identifies honeypots, evil twins, WEP networks, suspicious SSIDs |
| **Encryption Analysis** | Classifies WPA2/WPA3/WEP/Open with color-coded badges |
| **Signal Strength** | Shows signal percentage for each network |
| **AI-Powered Insights** | Provides security posture analysis and recommendations |
| **Dark Theme Dashboard** | Professional, modern UI with real-time updates |
| **Rate Limiting** | 15-second delay to respect free API limits |
